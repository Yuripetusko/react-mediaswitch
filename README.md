react-mediaswitch
=================

Choose your DOM based on media queries


```
<MediaSwitch>
  <MediaCase media="screen and (max-width: 400px)">
    <div>So tiny!</div>
  </MediaCase>
  <MediaCase media="screen and (max-width: 800px)">
    <div>A little bigger</div>
  </MediaCase>
</MediaSwitch>
```
